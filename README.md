# lekcje_python

Lekcje z podstaw programowania dla studentów, zapisane bezpośrednio w repozytorium.

## Co jest w repozytorium

- `notebooks/lekcja_podstawy_python.ipynb` — gotowa lekcja w formacie Jupyter/Colab.

## Uruchamianie w Google Colab

1. Wejdź na [Google Colab](https://colab.research.google.com/).
2. Wybierz **File → Open notebook → GitHub**.
3. Wklej URL tego repozytorium i otwórz plik:
   - `notebooks/lekcja_podstawy_python.ipynb`
