# lekcje_python

Zestaw 12 lekcji programowania w Pythonie, ukierunkowanych na **obsługę API** oraz **analizę danych**.

## Plan kursu (12 lekcji)

1. `notebooks/lekcja_01_python_pod_api_i_analize_danych.ipynb` — Wprowadzenie do Pythona pod API i analizę danych.
2. `notebooks/lekcja_02_funkcje_i_moduly_w_praktyce.ipynb` — Funkcje i moduły w praktyce danych.
3. `notebooks/lekcja_03_obsluga_plikow_csv_json.ipynb` — Praca z plikami CSV i JSON.
4. `notebooks/lekcja_04_wprowadzenie_do_requests_i_http.ipynb` — Wprowadzenie do HTTP i biblioteki `requests`.
5. `notebooks/lekcja_05_api_paginacja_i_filtrowanie.ipynb` — API: paginacja, filtrowanie i parametry zapytań.
6. `notebooks/lekcja_06_autoryzacja_i_bezpieczenstwo_api.ipynb` — Autoryzacja i bezpieczeństwo w API.
7. `notebooks/lekcja_07_pandas_podstawy.ipynb` — Pandas: podstawy analizy tabelarycznej.
8. `notebooks/lekcja_08_czyszczenie_i_transformacja_danych.ipynb` — Czyszczenie i transformacja danych.
9. `notebooks/lekcja_09_analiza_eksploracyjna_eda.ipynb` — Analiza eksploracyjna danych (EDA).
10. `notebooks/lekcja_10_wizualizacja_matplotlib_seaborn.ipynb` — Wizualizacja danych: matplotlib i seaborn.
11. `notebooks/lekcja_11_projekt_pipeline_api_do_raportu.ipynb` — Projekt: pipeline od API do raportu.
12. `notebooks/lekcja_12_dobre_praktyki_testy_i_automatyzacja.ipynb` — Dobre praktyki: testy, logowanie i automatyzacja.

## Uruchamianie w Google Colab

1. Wejdź na [Google Colab](https://colab.research.google.com/).
2. Wybierz **File → Open notebook → GitHub**.
3. Wklej URL tego repozytorium.
4. Otwórz wybrany plik z katalogu `notebooks/`.
